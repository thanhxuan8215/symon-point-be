## Description

Symon point APIs service repository.

## Installation

```bash
# install packages
$ npm install

# init database
$ docker-compose up
```

## Migration

```bash
# run migrations
$ npm run typeorm:run-migrations
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
